How to Import into Lightroom
1
Click Download .XMP Preset above to save the file
2
Open Lightroom Classic or Lightroom CC
3
Go to the Develop module and find the Presets panel on the left
4
Click the + icon → select Import Presets
5
Select the downloaded .XMP file and click Import
6
Find it under User Presets and click to apply!
